kubeset
===

동일한 User name을 사용하는 다양한 Kubernetes Cluster를 사용하는데 어려움이 있어 개발한 bash script입니다.

# 사용법
* ${HOME}/.kube/conf 파일 삭제 또는 이름변경
* ${HOME}/.bashrc 또는 .zshrc에 "source ${HOME}/.kubeconf" 명령어 추가
* ${HOME}/.kube 아래 kubeconfig 파일을 .kubeconfig 확장자로 저장
